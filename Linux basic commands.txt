Nmap:

Note: Nmap is a noisy scanner

nmap -sn [IP/range]
	-n=No.
	-s=Scan
		=No Port Scan
	-scan all networks in IP range and all live hosts open

nmap -sS [IP/range]
	-SYN scan or Stealth scan
	-Port Scan, TCP protocol for SYN scan
	-can be executed by default

nmap -sU [IP/range] or nmap -sU -T5 [IP/range]
	-port scan for UDP Protocol
	-UDP scan is slow due to its connectionless nature
	-use [-T5 timing option] to tweak it and make it faster
	-firewall blocks T5 on internet address, use T2 when scanning internet IP addresses

nmap --top-ports [number of ports(1000 is the default)] -T5 [IP/range]
	-used to scan range of ports you want to scan

nmap -p [port number] -T5 [IP/range]
	-used to scan the specified port or target port.

nmap -p- -T5 [IP/range]
	-used to include all target ports available/open. 

nmap -p- -T5 -sV [IP/range]
	[-sV Option]
	-used to tell the version of the scan
	s=scan, V=version
	-usually called the "version scan"
	nmap -p [port number1, port number2] -T5 -sV [IP/range]
		-speed the scanning for version scan
	
	Nmap Version Intensity:
	Highest=9, Default=7
	--version-intensity (default value: 7)
	--version-light (default value: 2)
	--version-all (default value: 9[it will scan all versions])
	
	Syntax:
	nmap -p [port number] -T5 -sV --version-intensity [IP/range]

nmap -A -v -v [IP/range]
	-gives a lot of information, even SO in some cases

nmap -PU [IP/range]
	-explains every IP Address

nmap -sV -O -T5 [IP/range]
	-displays the OS versions of devices in the network

NSE Script

A.
location: /usr/share/nmap/scripts/
-script [NSE script name]
	-NSE script option for nmap scanning
	Syntax: nmap -p [port number] --script [NSE script_name] [IP/range]
	
	Note: Add -sV option (version scan) for more detail/information
B.
-sC (sript scan)
	Syntax: nmap -sV -sC [IP/range]
	-doesn't need a port number to run a scan
	-target multiple port numbers (all TCP open ports)
	-has a low number of false positives (false positive means a false vulnerability)
	-less intrusive on the target system
	Note:
	     Some categories could bring your target down like the DOS category.

C.
-A 
     -execute the default script scan
    Option will be use:
	-Version scan
	-Syn TCP scan
	-Default NSE script scan
	-Operating System scan
	-Network traceroute
	
	Syntax: nmap -A [IP/range]

* (wildcard)
	-another way yo target multiple scripts in the same category
	
	Syntax: nmap -sV -p [port number] --script smb-vuln* [IP/range]
